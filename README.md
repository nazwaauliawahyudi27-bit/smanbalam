# smanbalam
WebGIS Persebaran SMA Negeri di Kota Bandar Lampung merupakan aplikasi berbasis web yang menampilkan persebaran 17 SMA Negeri di Kota Bandar Lampung secara interaktif menggunakan Leaflet, OpenStreetMap, dan data GeoJSON. Aplikasi ini menyediakan informasi lokasi sekolah untuk mendukung akses informasi pendidikan.
